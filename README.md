# Community Portal Project

## Project Overview

This project is a Community Portal for viewing and signing up to future events. It has 5 pages that includes an home, about, events, contact and thank you page.

## Technologies used

1. NodeJS
2. Express.JS
3. EJS
4. CSS
5. Github
6. Nodemon
